# go_bin_uninstaller
GOBINにインストールされている実行ファイルを削除する

## インストール

```
go install github.com/kznagamori/go_bin_uninstaller@latest
```

## 使い方

```go_bin_uninstaller [binaries...] [flags]
go_bin_uninstaller [binaries...] [flags]
```

| フラグ | 短縮フラグ | 概要                                                       |
| ------ | ---------- | ---------------------------------------------------------- |
| --list | -l         | インストールされているツール（実行ファイル）一覧を表示する |
